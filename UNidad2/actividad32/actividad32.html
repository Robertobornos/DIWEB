Propiedades de position en CSS

Position: static

Este es el comportamiento por defecto. El elemento sigue el flujo normal del documento y no se puede mover con top, left, etc.

Position: relative

Este elemento se mueve respecto a su posición original usando top y left.

Position: absolute

Este elemento se posiciona respecto al contenedor principal (que tiene position: relative).

Position: sticky

Este elemento se queda "pegado" en la parte superior al hacer scroll.

Position: fixed

Siempre visible en pantalla aunque hagas scroll.
