# COMPONENTES ESTRUCTURALES DE HTML5 (Versión Ampliada)

| Componente                   | Usos                                                                 | Etiquetas HTML5            | Ejemplos                                                                 |
|-----------------------------|----------------------------------------------------------------------|-----------------------------|-------------------------------------------------------------------------|
| Encabezado del documento    | Metadatos, título, enlaces a estilos y scripts                       | `<head>`                    | Presente en todos los sitios web                                        |
| Cuerpo                      | Todo el contenido visible de la página                               | `<body>`                    | Presente en todas las webs                                              |
| Cabecera                    | Logo, título del sitio, menú principal, buscador                     | `<header>`                  | Wikipedia usa `<header>` para su logo y buscador                        |
| Contenido principal         | Contenido relevante único de la página                               | `<main>`                    | GitHub usa `<main>` para mostrar repositorios y perfiles                |
| Pie de página               | Derechos, enlaces legales, contacto, info corporativa                | `<footer>`                  | Apple usa `<footer>` para enlaces corporativos y legales                |
| Barra lateral               | Información complementaria: widgets, enlaces, publicidad             | `<aside>`                   | The Guardian usa `<aside>` para noticias relacionadas                   |
| Navegación                  | Enlaces de navegación entre secciones o páginas                      | `<nav>`                     | BBC usa `<nav>` para su menú superior                                   |
| Sección                     | Bloques temáticos dentro del contenido                               | `<section>`                 | MDN Web Docs usa `<section>` para agrupar documentación                 |
| Artículo                    | Contenido independiente: noticias, blogs, publicaciones              | `<article>`                 | Medium usa `<article>` para cada entrada de blog                        |
| División genérica           | Agrupar contenido sin semántica específica                           | `<div>`                     | Usado en prácticamente todas las webs                                   |
| Encabezados de texto        | Títulos jerárquicos del contenido                                    | `<h1>`–`<h6>`               | Titulares de periódicos o blogs                                         |
| Párrafo                     | Texto general                                                        | `<p>`                       | Cualquier artículo o página informativa                                 |
| Lista no ordenada           | Listas de elementos sin orden específico                             | `<ul>` + `<li>`             | Listas de características de productos                                  |
| Lista ordenada              | Listas numeradas                                                     | `<ol>` + `<li>`             | Rankings, pasos de instrucciones                                        |
| Lista de definiciones       | Glosarios, diccionarios, pares término–definición                    | `<dl>`, `<dt>`, `<dd>`      | Documentación técnica                                                   |
| Figura                      | Imágenes con pie, diagramas, gráficos                                | `<figure>` + `<figcaption>` | Artículos científicos o técnicos                                        |
| Bloque de cita              | Citas textuales largas                                               | `<blockquote>`              | Blogs, artículos de opinión                                             |
| Línea temática              | Separación visual de secciones                                       | `<hr>`                      | Artículos largos                                                        |
| Enfasis / importancia       | Resaltar palabras o frases                                           | `<em>`, `<strong>`          | Textos explicativos                                                     |
| Código                      | Mostrar fragmentos de código                                         | `<code>`, `<pre>`           | Documentación de programación                                           |
| Formulario                  | Recoger datos del usuario                                            | `<form>`                    | Formularios de registro o contacto                                      |
| Campos de formulario        | Entradas de texto, botones, selectores                               | `<input>`, `<textarea>`, `<button>`, `<select>` | Formularios de login o búsqueda |
| Tabla                       | Datos estructurados                                                  | `<table>`, `<tr>`, `<td>`, `<th>` | Webs de estadísticas o comparativas |
| Contenido multimedia        | Vídeos, audio, controles multimedia                                  | `<video>`, `<audio>`        | Reproductores integrados                                                |
| Contenido embebido          | Integrar contenido externo                                           | `<iframe>`                  | Vídeos de YouTube, mapas de Google                                     |
| Contenedor interactivo      | Widgets, elementos dinámicos                                         | `<details>` + `<summary>`   | FAQs desplegables                                                       |
| Etiqueta genérica de texto  | Agrupar texto sin semántica                                          | `<span>`                    | Estilos en línea                                                        |
